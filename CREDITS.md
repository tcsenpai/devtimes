# Credits

The Dev Times stands on the shoulders of many. With thanks:

## Data

- **[daily.dev](https://daily.dev)** — every story on the front page, every tag,
  every byline. The whole reason this paper has anything to print. Used via the
  [Public API v1](https://docs.daily.dev/docs/plus/public-api).
- **[Tyler Vigen — Spurious Correlations](https://www.tylervigen.com/spurious-correlations)**
  — the inimitable source of the *Statistician's Corner* on page two. Wholly
  invented chart data, scrupulously real chart. Inspired by the
  [`spurelations`](https://pypi.org/project/spurelations/) Python package by
  [@tcsenpai](https://github.com/tcsenpai).

## Type

- **[Playfair Display](https://fonts.google.com/specimen/Playfair+Display)** —
  body headlines.
- **[Old Standard TT](https://fonts.google.com/specimen/Old+Standard+TT)** —
  body type.
- **[UnifrakturCook](https://fonts.google.com/specimen/UnifrakturCook)** —
  the blackletter masthead.

All from Google Fonts; all under the SIL Open Font License.

## Mechanics

- **[react-pageflip](https://github.com/Nodlik/react-pageflip)** and
  **[page-flip / StPageFlip](https://github.com/Nodlik/StPageFlip)** — the 3D
  page-turn animation that makes the whole thing feel like a real newspaper.
- **[Next.js](https://nextjs.org)** — App Router, standalone build, the works.
- **[Tailwind CSS](https://tailwindcss.com)** — styling, sparingly.
- **[zod](https://zod.dev)** — schema parsing for daily.dev payloads.
- **[bun:sqlite](https://bun.sh/docs/api/sqlite)** /
  **[better-sqlite3](https://github.com/WiseLibs/better-sqlite3)** — the
  Letters-to-the-Editor mailbag.
